認證說明

大部分 API 端點需要 JWT Token 認證。在請求標頭中加入：

Authorization: Bearer <your-jwt-token>

API 概覽

Service Booking API 是一個基於 TypeScript + Node.js + Koa + SQLite 的現代化服務預約管理後端 API 系統。

回應格式

所有成功的回應都遵循以下格式：

{
  "success": true,
  "data": { /* 回應資料 */ },
  "message": "操作成功",
  "timestamp": "2024-01-01T00:00:00.000Z"
}

錯誤回應格式：

{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "錯誤訊息",
    "details": [
      {
        "field": "欄位名稱",
        "message": "欄位錯誤訊息"
      }
    ]
  },
  "timestamp": "2024-01-01T00:00:00.000Z",
  "path": "/api/path"
}

認證端點

POST /auth/register
用戶註冊

請求範例：

curl -X POST http://localhost:3000/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "password123",
    "name": "張三"
  }'
201 Created 400 Bad Request 409 Conflict
POST /auth/login
用戶登入

請求範例：

curl -X POST http://localhost:3000/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "password123"
  }'
200 OK 400 Bad Request 401 Unauthorized
POST /auth/logout Auth Required
用戶登出
200 OK 401 Unauthorized
POST /auth/refresh Auth Required
更新 Token
200 OK 401 Unauthorized

服務管理

GET /services
獲取服務列表

查詢參數：

  • page - 頁碼 (預設: 1)
  • limit - 每頁筆數 (預設: 10)
  • sortBy - 排序欄位 (name, price, order, createdAt)
  • sortOrder - 排序方向 (asc, desc)

請求範例：

curl -X GET "http://localhost:3000/services?page=1&limit=10&sortBy=order"
200 OK
GET /services/:id
獲取服務詳情

請求範例：

curl -X GET http://localhost:3000/services/123e4567-e89b-12d3-a456-426614174001
200 OK 400 Bad Request 404 Not Found
POST /services Auth Required
創建新服務

請求範例：

curl -X POST http://localhost:3000/services \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <your-jwt-token>" \
  -d '{
    "name": "網站開發服務",
    "description": "提供專業的網站開發服務",
    "price": 50000,
    "showTime": 60,
    "order": 1,
    "isPublic": true
  }'
201 Created 400 Bad Request 401 Unauthorized 403 Forbidden
PUT /services/:id Auth Required
更新服務
200 OK 400 Bad Request 401 Unauthorized 403 Forbidden 404 Not Found
DELETE /services/:id Auth Required
刪除服務
200 OK 400 Bad Request 401 Unauthorized 403 Forbidden 404 Not Found
